Questions Fréquentes

Quelques réponses aux questions les plus posées

Comment puis-je investir dans un service ?

Pour investir dans un service, rendez-vous dans la section "Services", choisissez le service qui vous intéresse, cliquez sur "Acheter" et suivez les instructions de paiement. Vous devrez ensuite contacter l'administrateur via WhatsApp pour obtenir votre ID de transaction et finaliser votre achat.

Quel est le délai de traitement des retraits ?

Les retraits sont généralement traités dans un délai de 24 à 48 heures. Vous devez d'abord contacter l'administrateur via WhatsApp pour demander un retrait. Une fois que vous avez reçu votre ID de retrait, vous pouvez finaliser l'opération dans votre espace personnel.

Comment fonctionne le système de parrainage ?

Le système de parrainage vous permet de gagner 500 FCFA pour chaque personne que vous parrainez et qui achète un service. Vous recevez un lien unique à partager avec vos proches. Lorsque quelqu'un s'inscrit via votre lien et achète un service, vous recevez automatiquement votre commission.
